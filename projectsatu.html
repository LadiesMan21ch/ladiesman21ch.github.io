Aplikasi Kamus Tetun-Indonesia-Inggris

Aplikasi Kamus Tetun-Indonesia-Inggris adalah project Tugas Akhir (Skripsi) saya

Dalam project tugas akhir (Skripsi) saya ini saya mengembangkan sebuah aplikasi berbasis mobile Android dengan menggunakan Tools Android Studio Beta yang baru dirilis oleh Google

Gambar

Slide Menu Pilihan Terjemahan

Di dalam layout (Activity) 1 aplikasi kamus ini akan menampilkan tab slide untuk memilih menu terjemahan bahasa Tetun-Indonesia, Indonesia-Tetun, Tetun-Inggris, Inggris-Tetun dan pilihan menu Tentang (About).

Terjemahan (Translator)

Setelah membuka Tab Slide maka user akan memilih salah satu menu dan akan menampilkan seperti pada gambar di samping, disini user akan menerjemahkan kata dari pilihan menu yang di pilih di Tab Slide tadi.
